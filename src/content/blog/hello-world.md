Welcome to my blog! This is a fresh start for my personal website, built with [Astro](https://astro.build) and styled with Tailwind CSS.

## What to Expect

I plan to write about:

- **Ruby development** - Tips, tricks, and lessons learned from building Ruby applications
- **Software engineering** - Best practices, architecture decisions, and problem-solving approaches
- **Tools and workflows** - The tools I use and how I stay productive

## Why Astro?

As a Ruby developer, I wanted something that felt familiar but also let me explore modern frontend tooling. Astro hits that sweet spot:

- Write content in Markdown
- Fast static site generation
- Great developer experience
- Easy to deploy anywhere
